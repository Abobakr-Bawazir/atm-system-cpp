# ATM System (C++)

A console-based ATM System developed in C++.

This project was built as part of **Course 8** in my C++ learning journey.

The application simulates a real ATM machine where clients can log in using their account number and PIN, perform banking transactions, and manage their account balance using text file storage.

The project was developed while learning from **Dr. Mohammad Abu Hadhoud**. The implementation includes my own practice and modifications based on the concepts taught throughout the course.

---

## Features

### Authentication
- Login using Account Number
- PIN verification
- Logout

### ATM Operations
- Quick Withdraw
- Normal Withdraw
- Deposit Money
- Check Account Balance

### Quick Withdraw Options
- 20
- 50
- 100
- 200
- 400
- 600
- 800
- 1000

### Transaction Validation
- Prevent withdrawing more than the available balance
- Validate withdrawal amounts
- Confirm transactions before updating balances

### File Handling
- Load client information from `Clients.txt`
- Save updated balances to `Clients.txt`
- Read and write client records using text files
